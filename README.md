# 🐍 Snake Water Gun Game (CLI Version)

A simple command-line Snake Water Gun game built using Python.  
The game includes colored output, score tracking, and multiple rounds.

## 🚀 Features

- 🎯 5-round gameplay
- 🎨 Colored terminal output using Colorama
- 📊 Score tracking system
- ❌ Invalid input handling
- 🧠 Random computer choice logic
- 🐍 Clean and structured game logic


## 🛠 Technologies Used
- Python
- Random module
- Colorama library


## 🎮 How to Play

1. Run the Python file:
2. Enter your choice:
- `s` for Snake 🐍
- `w` for Water 💧
- `g` for Gun 🔫
3. The computer randomly selects its choice.
4. The winner is decided based on:
- Snake beats Water
- Water beats Gun
- Gun beats Snake
5. Final score is displayed after 5 rounds.
